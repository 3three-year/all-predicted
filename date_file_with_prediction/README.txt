================================================================================
使用方法
================================================================================

方法1: 临时替换（推荐）
--------------------------------------
1. 备份原始date_file目录:
   xcopy date_file date_file_backup /E /I /H

2. 复制新文件到date_file目录:
   copy date_file_with_prediction\*.csv date_file\

3. 运行测试（只测试这2天）:
   python train_simplified.py --mode test --num_days 2

4. 恢复原始文件:
   xcopy date_file_backup date_file /E /I /H /Y
   rmdir /s /q date_file_backup

方法2: 修改环境配置指向新目录
--------------------------------------
1. 打开 rural_env.py

2. 找到数据加载部分（约第203行）:
   data_dir = os.path.join(project_root, 'date_file')

3. 临时修改为:
   data_dir = os.path.join(project_root, 'date_file_with_prediction')

4. 运行测试:
   python train_simplified.py --mode test --num_days 2

5. 测试完成后恢复修改
