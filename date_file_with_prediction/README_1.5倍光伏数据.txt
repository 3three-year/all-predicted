使用方法
================================================================================

方法1: 修改rural_env.py使用1.5倍数据
--------------------------------------
在rural_env.py中修改光伏数据文件路径（约第207行）:

self.pv_data = pd.read_csv(
    os.path.join(data_dir, '按日累计-台区R1-总光伏出力_1.5倍_含预测.csv'),
    encoding='utf-8'
)

方法2: 临时替换文件
--------------------------------------
# 备份当前使用的文件
copy date_file_with_prediction\按日累计-台区R1-总光伏出力_含预测.csv date_file_with_prediction\按日累计-台区R1-总光伏出力_含预测_backup.csv

# 运行测试
