# 光伏预测数据处理流程说明

## 概述
本文档说明如何将光伏预测数据整合到全年数据中，并转换为强化学习环境所需的格式。

## 处理流程

### 步骤1: 替换预测数据到全年数据
**脚本**: `replace_pv_prediction_to_yearly.py`

**输入文件**:
- `DATE_FILE_predict/处理后的全年光伏数据.xlsx` (35040行×2列，时间序列格式)
- `DATE_FILE_predict/单日数据_2022-09-16.xlsx` (96个时间点的预测数据)
- `DATE_FILE_predict/单日数据_2022-09-25.xlsx` (96个时间点的预测数据)

**处理过程**:
1. 读取全年光伏数据（时间序列格式，每15分钟一行）
2. 读取9月16日和9月25日的预测数据
3. 将这两天的96个时间点替换为预测值
4. 其他363天保持原始数据不变

**输出文件**:
- `date_file_with_prediction/处理后的全年光伏数据_含预测.xlsx` (35040行×2列)
- `date_file_with_prediction/按日累计-台区R1-总光伏出力_扩大2.0倍_含预测.csv` (临时文件)

**验证结果**:
- ✓ 9月16日: 96个时间点，平均34.49 kW，数据一致性验证通过
- ✓ 9月25日: 96个时间点，平均34.65 kW，数据一致性验证通过

---

### 步骤2: 转换为按日累计格式
**脚本**: `convert_timeseries_to_daily.py`

**输入文件**:
- `date_file_with_prediction/处理后的全年光伏数据_含预测.xlsx` (35040行×2列)

**处理过程**:
1. 读取时间序列格式的数据
2. 按日期分组，每天96个时间点
3. 将每天的96个时间点转换为一行（P1-P96列）
4. 生成365行×97列的按日累计格式

**输出文件**:
- `date_file_with_prediction/按日累计-台区R1-总光伏出力_含预测.csv` (365行×97列)

**数据格式**:
```
日期,P1,P2,P3,...,P96
2022-01-01,0.0119,0.0076,0.0015,...,0.005
2022-01-02,0.0101,0.0054,0.00585,...,0.0109
...
2022-09-16,0.5574,0.5407,0.5262,...,0.5484  (预测数据)
...
2022-09-25,0.5359,0.5212,0.5064,...,0.5388  (预测数据)
...
2022-12-31,...
```

**验证结果**:
- ✓ 格式: 365行 × 97列（1个日期列 + 96个P列）
- ✓ 与参考文件格式一致
- ✓ 9月16日和9月25日的预测数据正确嵌入
- ✓ 其他天数保持原始数据

## 最终生成的文件

### 1. 按日累计-台区R1-总光伏出力_含预测.csv
- **格式**: 365行 × 97列
- **用途**: 直接用于强化学习环境（rural_env.py）
- **内容**: 
  - 第1列: 日期（2022-01-01 至 2022-12-31）

- **格式**: 35040行 × 2列
- **用途**: 中间文件，用于数据分析和可视化
- **内容**:
  - 第1列: Timestamp（时间戳，每15分钟一行）
  - 第2列: Power（光伏出力，kW）

### 3. README_光伏预测数据.txt
- **用途**: 使用说明文档
