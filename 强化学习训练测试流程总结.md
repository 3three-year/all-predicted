## 🎯 两个阶段的区别与联系

| 维度 | **第一阶段：历史数据测试** | **第二阶段：预测数据测试** |
|------|--------------------------|--------------------------|
| **数据来源** | 真实历史数据（ground truth） | 预测模型输出（有误差） |
| **数据准确性** | 100%准确 | 有预测误差（MAPE约3-15%） |
| **测试目的** | 验证agent算法有效性 | 验证端到端系统实用性 |
| **应用场景** | 算法研究、性能对比 | **实际部署、日前调度** |
| **环境模式** | `mode='test'` | **需要添加`mode='forecast'`** |
| **agent状态** | 训练完成，权重固定 | 训练完成，权重固定（同一个agent） |
| **重要性** | ⭐⭐⭐ 必须通过 | ⭐⭐⭐⭐⭐ 最终目标 |

### 关键联系：
- **第一阶段是基础**：必须先在历史数据上验证agent能学会有效策略
- **第二阶段是目标**：最终要在预测数据上应用，这才是实际场景
- **使用同一个agent**：第二阶段使用第一阶段训练好的agent

---

## 💡 为什么需要两个阶段？

### 如果只做第一阶段（历史数据测试）：
❌ **问题**：
- 只知道agent在完美数据下表现好
- 不知道agent对预测误差的鲁棒性
- 无法验证"预测+调度"系统的实际可行性
- 无法评估预测误差对调度效果的影响

### 如果只做第二阶段（预测数据测试）：
❌ **问题**：
- 没有训练好的agent（需要第一阶段训练）
- 无法区分是agent问题还是预测问题
- 缺乏性能基准对比

### 两个阶段结合：
✅ **优势**：
- 第一阶段：验证agent在理想条件下的最优性能
- 第二阶段：验证agent在实际条件下的鲁棒性
- 可以量化预测误差对调度效果的影响
- 提供完整的系统可行性验证

### 步骤2: 创建预测数据加载脚本

```python
# load_forecast_data.py

import numpy as np
import pandas as pd

def load_forecast_results(date1='09-15', date2='10-15'):
    """加载两天的预测结果"""
    
    # 从预测结果CSV中提取两天的数据
    pv_results = pd.read_csv('results/PV_Forecast/pv_prediction_results.csv')
    tb_results = pd.read_csv('results/TaiquB/optimized_prediction_results.csv')
    r1_results = pd.read_csv('results/TaiquR1_Original/optimized_prediction_results.csv')
    
    # 提取两天的数据（样本34和64对应09-15和10-15）
    # 每个样本96个点
    pv_day1 = pv_results['Predicted'][33*96:(33+1)*96].values
    pv_day2 = pv_results['Predicted'][63*96:(63+1)*96].values
    
    tb_day1 = tb_results['Predicted'][33*96:(33+1)*96].values
    tb_day2 = tb_results['Predicted'][63*96:(63+1)*96].values
    
    r1_day1 = r1_results['Predicted'][30*96:(30+1)*96].values
    r1_day2 = r1_results['Predicted'][60*96:(60+1)*96].values
    
    # 合并两天
    forecast_data = {
        'pv': np.concatenate([pv_day1, pv_day2]),      # (192,)
        'load': np.concatenate([tb_day1, tb_day2]),    # (192,)
        'ev': np.concatenate([r1_day1, r1_day2]),      # (192,)
        'dates': ['2022-09-15', '2022-10-15']
    }
    
    return forecast_data
```

### 步骤3: 创建预测数据测试脚本

```python
# test_on_forecast.py

import numpy as np
import torch
from rural_env import CountrysideEnv
from DRL import DuelingDDQN
from load_forecast_data import load_forecast_results
from utils import plot_optimized_results

def test_on_forecast_data():
    """在预测的两天数据上测试训练好的agent"""
    
    # 1. 加载预测数据
    print("加载预测数据...")
    forecast_data = load_forecast_results(date1='09-15', date2='10-15')
    
    # 2. 创建预测数据环境
    print("创建预测数据环境...")
    env = CountrysideEnv(
        algo="dueling_dqn",
        num_days=2,  # 两天
        mode='forecast',
        forecast_data=forecast_data
    )
    
    # 3. 加载训练好的agent
    print("加载训练好的DDQN模型...")
    agent = DuelingDDQN(
        alpha=8e-4,
        state_dim=env.state_dim,
        action_dim=env.action_dim,
        fc1_dim=128,
        fc2_dim=128,
        ckpt_dir='checkpoints/DDQN',
        gamma=0.95,
        tau=0.01
    )
    agent.load_models()  # 加载训练好的权重
    
    # 4. 在预测数据上运行agent
    print("开始日前调度决策...")
    state = env.reset()
    done = False
    
    # 记录数据
    actions_log = []
    soc_log = []
    loads_log = []
    pvs_log = []
    rewards_log = []
    
    step = 0
    total_reward = 0
    
    while not done:
        # Agent选择动作（不探索，使用学到的策略）
        action = agent.choose_action(state, isTrain=False)
        
        # 环境执行动作
        next_state, reward, done, info = env.step(action)
        
        # 记录数据
        actions_log.append(action)
        soc_log.append(env.battery_soc)
        loads_log.append(env.load_demand)
        pvs_log.append(env._get_pv_output())
        rewards_log.append(reward)
        
        total_reward += reward
        state = next_state
        step += 1
        
        if step % 96 == 0:
            print(f"  完成第{step//96}天调度，累计奖励: {total_reward:.2f}")
    
    print(f"\n✅ 预测两天调度完成！")
    print(f"   总奖励: {total_reward:.2f}")
    print(f"   平均每步奖励: {total_reward/step:.2f}")
    
    # 5. 保存结果
    np.save('results/Forecast_Scheduling/actions.npy', actions_log)
    np.save('results/Forecast_Scheduling/soc.npy', soc_log)
    np.save('results/Forecast_Scheduling/loads.npy', loads_log)
    np.save('results/Forecast_Scheduling/pvs.npy', pvs_log)
    np.save('results/Forecast_Scheduling/rewards.npy', rewards_log)
    
    # 6. 可视化结果
    print("\n生成可视化结果...")
    plot_forecast_scheduling_results(
        actions=actions_log,
        soc=soc_log,
        loads=loads_log,
        pvs=pvs_log,
        forecast_data=forecast_data,
        output_dir='results/Forecast_Scheduling'
    )
    
    return {
        'total_reward': total_reward,
        'actions': actions_log,
        'soc': soc_log,
        'loads': loads_log,
        'pvs': pvs_log
    }

if __name__ == '__main__':
    results = test_on_forecast_data()
```
